About The Course

Web Application Programming and Hacking (WAPH)

This course provides a comprehensive introduction to modern web development and security. It covers both front-end and back-end technologies, teaching how to build dynamic, data-driven web applications. A significant portion of the course is dedicated to understanding and mitigating common security vulnerabilities, such as Cross-Site Scripting (XSS), SQL Injection, and Cross-Site Request Forgery (CSRF), providing a practical foundation in defensive programming and ethical hacking principles.

I am excited to continue the course and seeing what new things I can learn.
